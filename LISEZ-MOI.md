# Le Montmorency — Site web

## Contenu du dossier
- `index.html` — le site (HTML + CSS + JS dans un seul fichier, propre, sans base64)
- `logo.jpg` — logo du restaurant
- `photo.jpg` — photo de la salle
- `video.mp4` — vidéo de fond du hero

## Pour reprendre ce projet dans Claude Code

1. Ouvrez l'onglet **Code** dans l'application Claude Desktop
2. Choisissez ce dossier (`montmorency-site`) comme projet
3. Vous pouvez ensuite demander à Claude par exemple :
   - "Ouvre index.html et explique-moi sa structure"
   - "Optimise la vidéo pour qu'elle soit plus légère"
   - "Ajoute une section galerie photo"
   - "Prépare ce projet pour un déploiement sur Netlify avec Git"

## Pour voir le site immédiatement

Double-cliquez sur `index.html` — il s'ouvre dans votre navigateur avec le logo,
la photo et la vidéo déjà fonctionnels (gardez les 4 fichiers dans le même dossier).

## Ce qui a déjà été fait
- Design adapté aux couleurs et au logo du Montmorency
- Vrai menu intégré (braisés, sauces, boissons) avec les bons prix
- Formulaire de réservation connecté à Netlify Forms (fonctionnera une fois déployé sur Netlify)
- Site responsive (s'adapte aux téléphones)
- SEO de base (meta tags, Schema.org, Open Graph)
- Téléphone et email cliquables
